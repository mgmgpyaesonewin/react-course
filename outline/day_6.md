# Day 6 (3 Hours): Deployment and Final Project

## Hour 16: Deploying with Vercel
*   **Introduction to Vercel:** What is Vercel? Benefits for frontend and full-stack applications.
*   **Deployment Workflow:**
    *   Connecting your GitHub repository to Vercel.
    *   Automatic deployments on Git pushes.
    *   Custom domains (brief mention).
*   **Environment Variables on Vercel:**
    *   How to securely manage sensitive keys (like Supabase API keys) for different environments (development, production).
*   **Building for Production:** Vercel's build process.
*   **Exercise:** Deploy your React + Supabase To-Do List application to Vercel. Ensure all Supabase functionality (CRUD, Auth) works correctly on the deployed version.

## Hours 17-18: Guided Project Development
*   **Final Project Kick-off:** Students will begin their independent final projects.
*   **Brainstorming & Planning:**
    *   Defining project scope, features, and user stories.
    *   Designing the Supabase database schema for their project.
    *   Planning React component structure.
*   **Implementation Support:** Instructor available for guidance, debugging, and best practices.
*   **Goal:** Have a basic functional prototype of their final project (React + Supabase, with authentication and data display).
