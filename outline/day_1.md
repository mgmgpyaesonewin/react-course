# Day 1 (3 Hours): Web Foundations

## Hour 1: Introduction to HTML
*   **Concepts:** Core structure of a web page, elements vs. tags, attributes.
*   **Essential Tags:**
    *   `<!DOCTYPE html>`, `<html>`, `<head>`, `<body>`
    *   Headings (`<h1>` to `<h6>`)
    *   Paragraphs (`<p>`)
    *   Links (`<a>` with `href` and `target`)
    *   Images (`<img>` with `src`, `alt`, `width`, `height`)
    *   Lists (`<ul>`, `<ol>`, `<li>`)
    *   Divisions (`<div>`) and Spans (`<span>`)
*   **Exercise:** Create a simple personal bio page (`index.html`) with your name, a short paragraph, a link to your favorite website, and an image of something you like.

## Hour 2: Introduction to CSS
*   **Concepts:** What is CSS? Why use it? Cascading, specificity, inheritance.
*   **Styling Fundamentals:**
    *   Inline, Internal, and External CSS.
    *   Selectors: Element, Class (`.`), ID (`#`).
    *   Basic Properties: `color`, `background-color`, `font-family`, `font-size`.
    *   The Box Model: `margin`, `padding`, `border` (understanding how elements take up space).
*   **Exercise:** Style your `index.html` bio page:
    *   Give the `<body>` a background color.
    *   Change the font and color of your name.
    *   Add padding and a border around your paragraph.
    *   Center your image.

## Hour 3: JavaScript Fundamentals
*   **Concepts:** What is JavaScript? How it makes web pages interactive.
*   **Variables:** `let`, `const`, `var` (brief mention of `var` for context).
*   **Data Types:** Strings, Numbers, Booleans, Arrays, Objects.
*   **Functions:** Declaring and calling functions, parameters, return values.
*   **Basic DOM Manipulation:**
    *   `document.getElementById()`
    *   `document.querySelector()`, `document.querySelectorAll()`
    *   `element.innerHTML`, `element.textContent`
    *   `element.style.<property>`
    *   `element.addEventListener('click', function)`
*   **Exercise:** Add interactivity to your `index.html` bio page:
    *   Add a button that, when clicked, changes the text content of your paragraph.
    *   Add another button that changes the background color of your page.
